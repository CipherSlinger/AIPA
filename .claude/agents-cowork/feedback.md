## 工作流 Canvas 模式（拖拽式可视化工作流编辑器）

**功能描述：**
为工作流（Workflow）新增 Canvas 模式，提供拖拽式可视化编辑与运行监控体验。

**编辑阶段：**
- 用户可在 Canvas 画布上直接拖入已定义的 Agent 节点
- 通过连线方式定义 Agent 之间的执行顺序与依赖关系
- 支持设置节点参数、条件分支、并行/串行执行路径

**运行阶段：**
- 给工作流分派任务后，可一键切换至 Canvas 视图
- 画布实时高亮当前正在执行的 Agent 节点
- 已完成节点、等待中节点、失败节点以不同颜色区分
- 支持点击节点查看该 Agent 的输入/输出详情

**交互要求：**
- Canvas 视图与列表视图可随时切换，不丢失状态
- 画布缩放、平移，适配复杂大型工作流
- 工作流执行日志可在 Canvas 侧边栏折叠展示
